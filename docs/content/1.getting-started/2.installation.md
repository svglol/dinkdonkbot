## Prerequisites

Before you begin, make sure you have:

- **Server Permissions**: You need the "Manage Server" permission in the Discord server where you want to add the bot
- **Channel Permissions**: The bot needs permission to send messages, and mention everyone in channels where it will post notifications

## Adding DinkDonk Bot to Your Server

### Step 1: Invite the Bot

Click the button below or visit the Discord App Directory to invite DinkDonk Bot:

:::u-button
---
color: primary
size: xl
to: https://discord.com/application-directory/1227866873220173824
trailing-icon: i-lucide-rocket
---
Add to Discord
:::

### Step 2: Select Your Server

1. A new window will open asking you to select a server
2. Choose the server where you want to add DinkDonk Bot
3. Click "Continue"

### Step 3: Review Permissions

DinkDonk Bot requires the following permissions to function properly:

- **Send Messages**: To post stream notifications and respond to commands
- **Mention Everyone**: To ping roles in notifications (if configured)
- **Manage Expressions**: To add emotes and stickers to your server

Review the permissions and click "Authorize" to complete the installation.

### Step 4: Complete the Verification

Complete any verification challenges (such as CAPTCHA) if prompted.

## Verifying the Installation

Once added, you should see DinkDonk Bot appear in your server's member list. The bot will be online and ready to use.

To verify the bot is working correctly:

1. In any channel, type `/help`
2. If DinkDonk Bot responds with a help message, the installation was successful!
