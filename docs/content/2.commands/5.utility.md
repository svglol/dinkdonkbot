## Commands Overview

Useful utilities for server management and community engagement.

| Command | Description |
|---------|-------------|
| [`commands`](#commands) | List all available commands |
| [`dinkdonk`](#dinkdonk) | Get dinkdonked |
| [`help`](#help) | Show help for DinkDonk Bot |
| [`invite`](#invite) | Get an invite link to add the bot to your server |
| [`quickstart`](#quickstart) | Get started with DinkDonk Bot |
| [`randomemote`](#randomemote) | Post a random emote from the current server |
| [`time`](#time) | Get the current time for a location |
| [`timestamp`](#timestamp) | Create a Discord timestamp for a specific date/time |
| [`weather`](#weather) | Get the current weather for a location |

## Commands

List all available commands for DinkDonk Bot:
```
/commands
```

## DinkDonk

Get dinkdonked:
```
/dinkdonk
```

## Help

Show help information for DinkDonk Bot:
```
/help
```

## Invite

Get an invite link to add the bot to your server:
```
/invite
```

## Quickstart

Get started with DinkDonk Bot and learn the basics:
```
/quickstart
```

## Randomemote

Post a random emote from the current server:
```
/randomemote
```

## Time

Get the current time for a specific location:
```
/time <location>
```

## Timestamp

Create a Discord timestamp for a specific date/time and UTC offset:
```
/timestamp <date> <time> [utc-offset]
```

Discord will automatically display the timestamp in each user's local timezone.

## Weather

Get the current weather for a location:
```
/weather <location>
```
