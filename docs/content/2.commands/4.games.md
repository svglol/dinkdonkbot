## Commands Overview

Enjoy interactive games directly in your Discord server.

| Command | Description |
|---------|-------------|
| [`hangman`](#hangman) | Play a classic game of hangman |
| [`rps`](#rock-paper-scissors) | Challenge someone to rock, paper, scissors |
| [`roll`](#roll-dice) | Roll dice with customizable sides |
| [`coinflip`](#coinflip) | Flip a coin for heads or tails |

## Hangman

Start a game of hangman with your friends:
```
/hangman
```

Guess letters to reveal the hidden word before running out of attempts!

## Rock Paper Scissors

Challenge another user to rock, paper, scissors:
```
/rps <opponent>
```

Make your choice and see who wins!

## Roll Dice

Roll dice with a specified number of sides:
```
/roll [dice] [sides]
```

Default is a standard 6-sided die. Customize the number of sides and dice count for different game mechanics.

## Coinflip

Flip a coin to get heads or tails:
```
/coinflip
```

Perfect for making quick decisions!
