## Commands Overview

Currently only Twitch clips are supported. All clip commands start with `/clips twitch`

| Command | Description |
|---------|-------------|
| [`add`](#adding-a-twitch-streamer) | Add a Twitch streamer to subscribe to clips for |
| [`remove`](#removing-a-twitch-streamer) | Remove a Twitch streamer from receiving clip notifications |
| [`edit`](#editing-twitch-clips-settings) | Edit config for a Twitch clip subscription |
| [`list`](#listing-clips-subscriptions) | List all your Twitch clip subscriptions |
| [`help`](#help) | Get this help message for Twitch clip commands |

## Adding A Twitch Streamer
Subscribe to Twitch clips for a streamer:

```
/clips twitch add <streamer> <discord-channel>
```

## Removing A Twitch Streamer
Unsubscribe from Twitch clips for a streamer:
```
/clips twitch remove <streamer>
```

## Editing Twitch Clips Settings
Change the notification channel for a Twitch clip subscription:
```
/clips twitch edit <streamer> <discord-channel>
```
## Listing Clips Subscriptions
View all your Twitch clip subscriptions:
```
/clips twitch list
```
## Help
Get help information for Twitch clip commands:
```
/clips twitch help
```
