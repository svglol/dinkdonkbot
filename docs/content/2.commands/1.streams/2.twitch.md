## Commands Overview

All Twitch stream commands start with `/streams twitch`

| Command | Description |
|---------|-------------|
| [`add`](#adding-a-twitch-streamer) | Add a Twitch streamer to receive notifications for going live |
| [`remove`](#removing-a-twitch-streamer) | Remove a Twitch streamer alert subscription |
| [`edit`](#editing-twitch-settings) |  Edit config for a Twitch streamer alert |
| [`test`](#testing-notifications) | Preview notification appearance |
| [`details`](#viewing-streamer-details) | Show the current configuration for a Twitch streamer |

## Adding a Twitch Streamer

Add a new Twitch streamer to your notification list:

```
/streams twitch add <streamer> <discord-channel> [ping-role] [live-message] [offline-message] [cleanup]
```

### Parameters

- **streamer** (required) - The Twitch username of the streamer (case-insensitive)
- **discord-channel** (required) - The Discord channel where notifications will be posted
- **ping-role** (optional) - Role to mention when the streamer goes live
- **live-message** (optional) - Custom message for when streamer goes live
- **offline-message** (optional) - Custom message for when streamer goes offline
- **cleanup** (optional) - Auto-delete offline notifications (`true` or `false`)

## Removing a Twitch Streamer

Stop monitoring a Twitch streamer:

```
/streams twitch remove <streamer>
```

## Editing Twitch Settings

Modify settings for an existing Twitch streamer:

```
/streams twitch edit <streamer> [discord-channel] [ping-role] [remove-ping-role] [live-message] [offline-message] [cleanup]
```

### Parameters

- **streamer** (required) - The Twitch username of the notification we want to edit
- **discord-channel** (optional) - Change the notification channel
- **ping-role** (optional) - Add or change the ping role
- **remove-ping-role** (optional) - Remove the current ping role (`true`)
- **live-message** (optional) - Change the live notification message
- **offline-message** (optional) - Change the offline notification message
- **cleanup** (optional) - Enable/disable auto-cleanup (`true` or `false`)

## Testing Notifications

Preview what your notifications will look like:

```
/streams twitch test <streamer> [message-type] [multistream] [global]
```

### Parameters

- **streamer** (required) - The Twitch username of the notification you want to test
- **message-type** (optional) - Test `live` or `offline` notification (default: `live`)
- **multistream** (optional) - Test as multistream notification if it is setup (`true` or `false`)
- **global** (optional) - Display the message to everyone as if the streamer just went live (will ping people so use with caution) (`true` or `false`)

## Viewing Streamer Details

See all current settings for a specific streamer:

```
/streams twitch details <streamer>
```

## Customizing Messages

### Available Variables

Use these variables in your custom messages:

- `{{name}}` - The name of the streamer
- `{{url}}` - The URL for the stream
- `{{everyone}}` - @everyone mention
- `{{here}}` - @here mention
- `{{game}}`/`{{category}}` - The game or category of the stream (live only)
- `{{timestamp}}` - The time the stream started/ended
