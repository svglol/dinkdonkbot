## Overview

| Command | Description |
|---------|-------------|
| [`list`](#view-all-subscriptions) | List all your stream subscriptions across platforms |
| [`help`](#get-help) | Show help information for stream commands |

| [`kick`](/commands/streams/kick) | Manage Kick stream notifications |
| [`multistream`](/commands/streams/multistream) | Link Twitch and Kick accounts for unified notifications |

## View All Subscriptions

Display all your stream subscriptions across both Twitch and Kick:

```
/streams list
```

This shows:
- All Twitch streamers you're monitoring
- All Kick streamers you're monitoring
- Multistream connections and settings

## Get Help

Access the streams command help documentation:

```
/streams help
```

This displays:
- Available subcommands (twitch, kick, multistream)
- Quick usage examples
- Links to detailed documentation
- Common command patterns

Use this anytime you need a quick reference for stream commands.
