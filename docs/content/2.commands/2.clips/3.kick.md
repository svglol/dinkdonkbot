## Commands Overview
| Command | Description |
|---------|-------------|
| [`add`](#adding-a-kick-streamer) | Add a Kick streamer to subscribe to clips for |
| [`remove`](#removing-a-kick-streamer) | Remove a Kick streamer from receiving clip notifications |
| [`edit`](#editing-kick-clips-settings) | Edit config for a Kick clip subscription |
| [`list`](#listing-clips-subscriptions) | List all your Kick clip subscriptions |
## Adding A Kick Streamer
Subscribe to Kick clips for a streamer:
```
/clips kick add <streamer> <discord-channel>
```
### Parameters

::field-group
:::field{name="streamer" type="string" required}
The Kick username of the streamer
:::
:::field{name="discord-channel" type="text-channel" required}
The Discord channel where notifications will be posted
:::
::

## Removing A Kick Streamer
Unsubscribe from Kick clips for a streamer:
```
/clips kick remove <streamer>
```
::field-group
:::field{name="streamer" type="string" required}
The Kick username of the streamer
:::
::

## Editing Kick Clips Settings
Change the notification channel for a Kick clip subscription:
```
/clips kick edit <streamer> <discord-channel>
```

::field-group
:::field{name="streamer" type="string" required}
The Kick username of the streamer
:::
:::field{name="discord-channel" type="text-channel" required}
The Discord channel where notifications will be posted
:::
::
## Listing Clips Subscriptions
View all your Kick clip subscriptions:
```
/clips kick list
```
