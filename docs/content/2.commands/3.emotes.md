## Commands Overview

Manage custom emotes on your Discord server with simple commands.

| Command | Description |
|---------|-------------|
| [`add`](#adding-an-emote) | Add an emote from another Discord server or 7tv |
| [`help`](#help) | Get this help message for emote commands |

## Adding An Emote

Add a custom emote to your server:
```
/emote add <source>
```

**Supported sources:**
- Emote URL (from 7tv)
- Discord emoji (from another server)

## Context Menu Option

Right-click on any message containing an emote or sticker:
```
Apps -> Steal Emote/Sticker
```

This will instantly add the emote or sticker to your server.

## Help

Get help information for emote commands:
```
/emote help
```
